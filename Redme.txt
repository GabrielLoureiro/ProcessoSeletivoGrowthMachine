Passo a passo para a instalação do Cypress e rodar o teste:

    1- Clone o projeto: https://github.com/GabrielLoureiro/WebTestRobot.git
    2- Abra o Visual Studio Code
    3- Abra a pasta do projeto
    4- Depois de aberto o projeto no visual studio, abra um novo terminal utilizando crtl+shift+'
    5- Verifique que o caminho está correto para o projeto no terminal
    6- Digite o seguinte comando: npm install cypress
    7- Após instalar o Cypress digite o seguinte comando: npm run cypress
    8- Abrirá uma janela "Welcome to Cypress!"
    9- Após a janela se abrir clique em E2E Testing
    10- Escolha o navegador de sua preferencia e clique em Start E2E Testing in "nome do navegador escolhido"
    11- Em seguida abrirá outra janela contendo todos os testes a serem startados.
    12- Escolha o teste a ser iniciado e clique no nome dele que automaticamente ele será iniciado.
